# Base Mainnet 发布清单

任何一项未完成，都不应广播主网部署交易。

## 合约与构建

- [ ] `pnpm install --frozen-lockfile` 成功。
- [ ] `pnpm run check` 全部通过。
- [ ] 独立审计已完成，结论和修复记录可公开核验。
- [ ] 待部署源码与审计版本、Git 提交完全一致。
- [ ] 使用 `production` build profile，优化器为 200 runs。

## 地址与资产保管

- [ ] `initialHolder` 是经过实际收发测试的多签钱包。
- [ ] 多签阈值、签名人和恢复流程已经确定。
- [ ] 至少两个人逐字符复核 `initialHolder`。
- [ ] 部署钱包仅保留足够支付 Gas 的 ETH。
- [ ] 私钥没有写入仓库、参数 JSON、聊天、截图或 CI。

## 代币与公开信息

- [ ] 名称 `Galaxy`、符号 `GXLY`、8 位小数和 2,000 万总量最终确认。
- [ ] 团队、社区、流动性和储备分配已公开。
- [ ] 如有锁仓承诺，使用经过审计的链上锁仓合约执行。
- [ ] 已完成商标、证券、税务和所在司法辖区的合规评估。

## 部署与验证

- [ ] 先在 Base Sepolia 使用最终参数完整演练。
- [ ] 主网 chain ID 确认为 `8453`。
- [ ] RPC 域名和区块浏览器由第二人独立核对。
- [ ] 部署后记录 Git 提交、交易哈希、合约地址、部署者和接收者。
- [ ] BaseScan 源码验证成功，编译器和优化器参数一致。
- [ ] 链上读取 `name`、`symbol`、`decimals`、`MAX_SUPPLY`、`totalSupply` 和接收者余额并留档。

## 交易与运营

- [ ] 流动性池的交易对、初始价格和投入数量已经书面确认。
- [ ] 流动性所有权或锁定方式已公开。
- [ ] 官网只公布唯一主网合约地址，并提供防假币提示。
- [ ] 建立异常监控、社区通知和事故响应负责人。
